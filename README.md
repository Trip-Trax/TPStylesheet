> Create dynamic stylessheet and use JSON to add styles to it with this lightweight module.

## Installation
Install with npm:
```shell
npm install ttstylesheet
```

## Usage
```javascript
import stylesheet from 'ttstylesheet';

const customStylesheet = new stylesheet();
```
